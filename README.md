## Работа через Pull Request
1. Создайте отдельную ветку для ваших изменений
2. Внесите изменения и сделайте коммит
3. Запушьте ветку в репозиторий
4. Создайте Pull Request через интерфейс GitHub
5. Добавьте описание и укажите связанные Issues
6. Назначьте ревьюеров
7. После одобрения выполните merge
